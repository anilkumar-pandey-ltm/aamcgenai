📊 BDD Test Scenario Generation - Phase Overview

High-Level 5-Phase Flow

Phase 1: Server Verification

  • ✅ Check Atlassian MCP Server
  • ✅ Check Context MCP Server
  • ✅ Verify 3 context files
  • ⏱️ Time: 2-3 seconds

Phase 2: Story Retrieval

  • 📥 Check local cache
  • 🔗 Fetch from JIRA API
  • 💾 Parse and save story
  • ⏱️ Time: 3-5 seconds

Phase 3: Context Loading

  • 📱 Application context
  • 🏗️ Domain model
  • 📋 Business rules
  • ⏱️ Time: 2-3 seconds

Phase 4: Scenario Generation

  • 📝 Plan coverage (A-G)
  • ✍️ Write Gherkin scenarios
  • 🔢 Create Examples tables
  • ⏱️ Time: 10-15 seconds

Phase 5: Validation & Save

  • ✔️ Validate coverage
  • 📊 Calculate quality score
  • 💾 Save feature file
  • ⏱️ Time: 3-4 seconds
